# 🧠 AI-Based Disease Diagnosis System (CLI)

## 📌 Overview
This project is a simple Machine Learning-based disease prediction system that helps in identifying possible diseases based on patient symptoms.

It is built as a CLI (Command Line Interface) application using Python and Scikit-learn.

---

## 🎯 Objective
To create a basic AI system that:
- Takes symptoms as input
- Predicts possible disease
- Demonstrates real-world use of Machine Learning in healthcare

---

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 📊 Dataset
The dataset contains symptoms such as:
- Fever
- Cough
- Headache
- Fatigue
- Nausea

Each symptom is represented as:
- 1 → Yes
- 0 → No
